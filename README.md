Fruit Store is a modern mobile E-commerce Android application designed to connect fruit buyers and sellers in real time.
It features a Buyer Panel and a Seller (Admin) Panel, both synced through Firebase for live updates, seamless ordering, and easy inventory management.

This app supports:

Firebase Authentication

Real-time Realtime Database

Image uploads

Geolocation via Google Maps API

Stripe API for online payments

Order history + order management

Admin CRUD panel for fruits, clients, and orders
